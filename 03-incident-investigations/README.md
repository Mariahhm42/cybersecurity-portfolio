# Incident Investigations

Each file in this folder is one completed investigation (from TryHackMe, LetsDefend, BTLO, or your own SIEM lab), written up using the template below.

## Write-up template
Copy this into a new file named `YYYY-MM-DD-short-title.md` for each investigation:

```
# Title

**Scenario:** What happened (1-2 sentences)
**Tools used:** e.g. Splunk, Wireshark, VirusTotal, Wazuh
**Investigation steps:** What you looked at, in order
**Finding:** What the threat/root cause was
**MITRE ATT&CK mapping:** Technique ID(s)
**Screenshots:** (embed or link)
**Lessons learned:**
```
